What Penny does

Everything you need, nothing you don't

A full personal finance suite that works entirely on-device — no subscriptions, no data collection, no internet required.

💸

Transaction Tracking

Record income and expenses with categories, notes, multi-currency support, receipt scanning, and full audit trail.

📊

Budgets

Create period-based budgets with per-category limits. Track actual vs. limit with progress bars and overspent alerts.

🎯

Savings Goals

Set targets with deadlines, track contributions, and see monthly savings required to hit your goal on time.

📈

Advanced Analytics

Cash flow analysis, budget variance reports, 3-month trend projections, and bar/line/pie charts — all offline.

🤝

Expense Splitting

Create split groups, divide expenses equally or with custom percentages, track balances, and settle debts. Auth-gated with contact-picker invites.

👥

Contacts & Associations

Search users by share ID (PENNY-XXXX), send association requests with accept/reject/block flow, in-app notification banners with unique sound effects via Supabase Realtime.

📷

Receipt Scanning

On-device ML Kit OCR extracts merchant, total, tax, date, and line items. Optional self-hosted PaddleOCR/docTR backend.

🌍

Multi-Currency

15 currencies with live EC InforEuro exchange rates, offline gzip cache, and automatic EUR-based conversion.

📲

QR Code Sharing

Share transactions via ECDSA P-256 signed QR codes with 24-hour expiry. Works entirely offline, no backend needed.

☁️

Cloud Sync

Optional Google Drive backup and restore. Auto-sync when connectivity resumes, with conflict count reporting.

🔁

Recurring Transactions

Configurable templates with daily/weekly/monthly/yearly recurrence. Backend scheduler runs at 2 AM.

🤖

AI Categorization

Rule-based auto-categorization with EXACT/CONTAINS/REGEX matching, 15 system defaults, and self-learning from your choices.

📄

Export & Reports

Generate PDF financial reports, export CSV/Excel, and take full JSON backups of all 19 tables.

🏦

Apple Wallet Import

Import transactions directly from Apple Wallet via FinanceKit (iOS 17.4+). MCC auto-categorization, duplicate detection, and merge/overwrite policies.

🔗

Supabase Cloud Sync

Optional Supabase backend with real-time sync, Edge Functions, and PostgreSQL — replacing 10 Spring Boot microservices with zero infrastructure.

🔔

In-App Notifications

Animated slide-down banners for contact requests, group invites, and workspace activity. Unique sound effects per notification type, powered by Supabase Realtime.

Privacy by design

Offline-first, always.

Your data lives on your device. Penny is fully functional without any network connection. Cloud features are strictly opt-in — nothing leaves your device by default.

  • ✓No account required — all core features work on first launch
  • ✓SQLite database stored locally via Drift (native & WASM)
  • ✓ECDSA P-256 private keys in FlutterSecureStorage, encrypted at rest
  • ✓Backend API calls guarded — release builds never call localhost
  • ✓Offline banner + connectivity guard on all network operations
  • ✓Receipt OCR runs on-device by default (ML Kit, no data sent)
